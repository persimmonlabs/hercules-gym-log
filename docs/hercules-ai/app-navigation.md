# Hercules App Navigation Guide

## Key Terminology

### WORKOUT
A collection of exercises that you perform together in one session.
Example: "Push Day" containing Bench Press, Shoulder Press, Tricep Dips

### PLAN / PROGRAM
A collection of workouts organized into a training program.
Example: "Push Pull Legs" program containing 6 workouts (Push A, Pull A, Legs A, Push B, Pull B, Legs B)

### SCHEDULE
A weekly or rotating configuration that determines which workout to do on which day.
Types: Weekly (specific days) or Rotation (cycle through workouts)

### SESSION
A completed workout that has been logged with actual sets, reps, and weights.

## Creating Content

### How to Create a Workout
1. Go to the Programs tab (bottom navigation)
2. Find the "My Workouts" section
3. Tap "Add Workout" button
4. Select exercises from the exercise library
5. Save the workout with a name

### How to Create a Plan/Program
There are multiple ways to create a plan:

**Option A — Create a Custom Plan (primary flow):**
1. Go to the Programs tab (bottom navigation)
2. Find the "My Plans" section
3. Tap the "Create Plan" button
4. Add WORKOUTS to the plan (you can select from your existing workouts)
5. Save the plan

**Option B — Add a Premade Plan from the Hercules Library:**
1. Go to the Programs tab
2. Find the "My Plans" section
3. Tap the "Add Plan" button
4. Browse and select a premade plan from the Hercules Library

**Option C — Use Hercules AI:**
- Ask Hercules AI to create a customized plan tailored to your goals (e.g., "Create me a 4-day upper/lower split")
- The AI will propose a plan with workouts and exercises for you to approve or modify

IMPORTANT: Plans contain WORKOUTS, not individual exercises. Users cannot add exercises directly to plans — they must first create workouts, then add those workouts to a plan. Make sure to create your workouts first if you haven't done so already.

### How to Start a Workout
1. From Dashboard: Tap "Today's Workout" if you have a schedule
2. From Programs tab: Tap on a workout in "My Workouts", then tap "Start"
3. From Calendar: Tap a scheduled workout day

### How to Log a Workout
There are three ways to log a workout:

**Option A — Live Session Tracking:**
1. Go to the Session tab (middle tab in bottom navigation)
2. Start a workout session and track your sets, reps, and weights in real time as you perform your workout

**Option B — Log a Past Session via Calendar:**
1. Go to the Calendar tab
2. Navigate to the date you want to log a workout for
3. Tap the date, then tap the "Log a session" button
4. Fill in the details of the workout (exercises, sets, reps, weights)

**Option C — Log via Hercules AI:**
- Ask Hercules AI to log a session for you (e.g., "Log that I did Push Day yesterday with Bench Press 3x10 at 135 lbs")
- Include details like exercises, sets, reps, and weights for accurate tracking

## App Sections

### Dashboard Tab
- Today's Workout card (if scheduled)
- Recent workouts summary
- Quick stats

### Programs Tab
- My Workouts: All your created workout templates
- My Plans: Your training programs
- My Schedule: Active workout schedule

### Calendar Tab
- Monthly view of workout history
- Tap dates to see or add sessions

### Performance Tab
- Analytic charts and insights
- Volume trends and training balance
- Personal records (PRs)
- Hercules AI can also provide information about your personal records — just ask in the chat (e.g., "What are my PRs?" or "What's my bench press PR?")

### Profile & Settings
- Access by tapping the profile icon at the top right of the Dashboard tab
- Contains user preferences: units (Imperial/Metric), theme (light/dark), haptic feedback
- Profile information: name, date of birth, gender, experience level, goals, equipment, training days

### How to Access Settings
1. Go to the Dashboard tab (home screen)
2. Tap the profile icon at the top right corner
3. This opens Profile & Settings where you can change units, theme, and other preferences

## Features That DO NOT Exist
- Setting sets/reps when creating plans (plans only contain workout references)
- Adding exercises directly to plans (must add workouts instead)
- Automatic exercise suggestions based on equipment (user selects manually)
